# tooltip in css

A light weight Tool tip using CSS only 

## Variables

```
--background-color:red;
```

## html

```
 <link rel="stylesheet" href="tooltip.css">
 <div class="tooltip" data-tool-tip="I am testing">TEST</div>

 .tooltip{
    --background-color:black;
 }
 ```
